Build a model that predicts target values from sequential market-state data.

  • Your ZIP must contain exactly one solution.py.
  • solution.py must define a PredictionModel class.
  • The scorer creates the model and calls predict(data_point) step by step.

The dataset contains anonymized market-state features.

  • datasets/train.parquet for local training.
  • datasets/valid.parquet for validation.
  • The hidden scorer runs the same model API on private parquet data.

When data_point.need_prediction is false, return None. When it is true, return a numeric array with shape (2,).

Final metric: mean weighted Pearson correlation.
